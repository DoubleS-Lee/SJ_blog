# Admin 2FA Guide

관리자 2단계 인증은 현재 앱 코드만으로 완전히 강제되지 않습니다.

이 프로젝트는 Google/Kakao OAuth 로그인 기반이라, 실제 보안 강제는 아래 두 군데에서 해야 합니다.

## 1. 관리자 계정 자체의 2단계 인증 켜기

관리자 권한을 가진 Google 계정, Kakao 계정에서 각각 2단계 인증을 활성화합니다.

- Google 계정: Google 계정 보안 설정 > 2단계 인증
- Kakao 계정: 카카오계정 보안 설정 > 2단계 인증

## 2. 관리자 계정 운영 원칙

1. 관리자 계정은 일반 회원용 계정과 분리합니다.
2. 관리자 계정은 개인용 브라우저 저장 로그인을 최소화합니다.
3. 관리자 계정 비밀번호 변경 이력이 있어야 합니다.
4. 관리자 권한은 꼭 필요한 계정에만 부여합니다.

## 3. 코드 레벨에서 이미 막고 있는 것

- `/admin` 접근은 서버에서 `users.is_admin`을 확인합니다.
- 관리자 업로드 API는 `is_admin` 확인 후에만 동작합니다.
